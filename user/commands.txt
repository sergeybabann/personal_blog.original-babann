---------- Актуальность версий ----------

проверить актуальность версий зависимостей на текущий момент времени
в файле package.json этой командой:

ncu -u --packageFile package.json


После запуска этой команды в терминале из корневого каталога проекта в файле 
package.json автоматически обновятся версии зависимостей на текущий момент времени.
В терминале отобразится разница старых и новых обновленных версий.
Соответственно файл package.json должен находиться в корневом каталоге проекта без
вложенностей. 

Затем для установки всех обновленнных модулей нужно запустить команду: 
npm install
